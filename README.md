# Personal Portfolio Website

## Overview

This is a responsive personal portfolio website built using HTML, CSS, and JavaScript. The website showcases my skills, projects, certifications, resume, and contact information.

## Features

* Responsive Design
* Smooth Scrolling Navigation
* Dark Mode Toggle
* Skills Section
* Projects Section
* Certifications Section
* Resume Download Button
* Contact Information

## Technologies Used

* HTML5
* CSS3
* JavaScript
* GitHub Pages
